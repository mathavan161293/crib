#Crib Hound

##Frontend

step 1: Run `npm install` to install the dependencies.
step 2: Run `npm start`
step 3: Run `npm run watch:css` to update the css from the tailwind

##Backend

step 1: Run `npm install` to install the dependencies.
step 2: Run `npm start`
